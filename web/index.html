About This Project

This project compares two popular graphics libraries (Skia and Raylib) by implementing the same animated robot face in both. The comparison includes native ARM64 builds for macOS and WebAssembly builds for browser performance testing.

Features: Animated blinking, interactive emotion controls, performance metrics, and side-by-side comparison.
